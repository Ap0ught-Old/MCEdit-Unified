directories.win32_utf8_argv()[source]¶

Uses shell32.GetCommandLineArgvW to get sys.argv as a list of UTF-8 strings.

Versions 2.5 and older of Python don’t support Unicode in sys.argv on Windows, with the underlying Windows API instead replacing multi-byte characters with ‘?’.

Returns None on failure.

Example usage:

>>> def main(argv=None):
...    if argv is None:
...        argv = win32_utf8_argv() or sys.argv
...
